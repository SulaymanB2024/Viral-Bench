# Creative Browser

Use official browser surfaces only for manual observation. Do not build scrapers,
automate login, bypass CAPTCHA or verification, evade rate limits, or collect
private account data.

## Workflow

1. Confirm `ALLOW_BROWSER_UI=true` before opening or controlling a browser UI.
2. Use TikTok Creative Center as a human research surface, not an automated data
   source.
3. Save only operator-observed notes: URL, platform, format, hook, visible
   metrics, visual pattern, CTA, and remake notes.
4. Add observations to local intake files or the trend database with citations.
5. Treat login, CAPTCHA, phone verification, age, region, and terms prompts as
   hard stop points for the operator.

## Boundaries

- No scraping, hidden API probing, or bulk extraction.
- No account creation, account verification, or posting.
- No cookies, session exports, credentials, recovery codes, or phone numbers in
  repo files.
