# WorthScan Posting QA Checklist

Complete this before a human manually posts any rendered package.

## Source And Visuals

- [ ] Every image is local and operator-approved.
- [ ] No seller contact details, private messages, addresses, credentials, or account identifiers are visible.
- [ ] Listing screenshots do not expose private account surfaces.
- [ ] Placeholder visuals have been replaced or clearly marked as not postable.

## Valuation

- [ ] Asking price is visible or stated.
- [ ] Estimated value is a range, not an unsupported exact claim.
- [ ] Comps were checked manually on the posting day.
- [ ] Confidence level matches comp quality.
- [ ] Disclaimer says the result is an estimate, not a guarantee or appraisal.

## Platform Boundary

- [ ] No scraping was used.
- [ ] No browser account action was automated.
- [ ] No paid provider call was made.
- [ ] No auto-posting was used.
- [ ] The human poster has approved the final caption, hashtags, script, and posting notes.

## After Manual Posting

- [ ] Create a metrics record with the final URL.
- [ ] Add a 1-hour snapshot.
- [ ] Schedule manual 24-hour, 72-hour, and 7-day snapshots.
