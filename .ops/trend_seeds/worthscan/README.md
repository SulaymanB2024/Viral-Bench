# WorthScan Trend Seeds

Store manually reviewed WorthScan trend examples here before importing them into
the local trend database.

Allowed inputs:

- Human-recorded Creative Center observations.
- Operator notes copied from visible UI facts.
- Local examples approved for reuse in WorthScan briefs.

Blocked inputs:

- TikTok, Instagram, or YouTube scraping.
- Hidden endpoint captures.
- Account automation, login bypass, or posting automation.
- Provider credentials or paid provider outputs.
